# A2O Sistemas — Protótipo do site

Protótipo navegável da home do site da **A2O Sistemas** (gestão logística integrada: TMS, Frota e Agro).

Arquivo único em HTML/CSS/JS, sem dependências além do Google Fonts. As imagens são de preenchimento (uso livre) e devem ser substituídas pelas oficiais.

## Estrutura

```
.
├── index.html            # página (servida pelo GitHub Pages)
└── assets/
    ├── logo.png
    ├── divisor-hero.svg
    ├── hero-image-2.png          # hero desktop
    ├── hero-image-mobile.png     # hero mobile
    ├── solucoes-imagens/         # imagens das soluções
    └── conteudo-imagens/         # imagens de blog e cases
```

## Rodar localmente

Basta abrir o `index.html` no navegador.

## Publicação

Hospedado via **GitHub Pages** (branch `main`, raiz do repositório).
